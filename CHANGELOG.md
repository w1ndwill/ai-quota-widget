# 更新记录

## v1.2.1 - 2026-07-12

- 支持独立启用或关闭 Codex、Claude Code、Antigravity 数据源，并让大窗布局随数据源状态自适应。
- 修复热力图在宽窗口下拉伸、与底部说明重叠的问题。
- 本地用量扫描改为版本化持久缓存：读取逻辑升级后会自动重建缓存，Codex/Claude 与 Antigravity 缓存相互隔离。
- 缓存写入使用原子替换；文件读取失败时不再展示过期统计；缓存默认保存到应用 `.userdata` 目录。

## v1.1.0 - 2026-07-12

- 修复限额刷新时被本地“纠错”逻辑错误覆盖为 0% 的问题，服务端额度为唯一来源。
- 缩短首次刷新等待：可选数据并行读取、重置卡缓存、延后历史图扫描。
- 暗色主题改为统一 CSS 色板，修复额度侧栏、圆环和文字对比不足。
- 重置卡主卡显示可用张数和最近到期时间；点击可查看全部卡片明细。
- 移除未使用的历史数据 IPC 桥接，保留统一历史读取接口。
